SUBIDA A GITHUB
1. Descomprimí este ZIP.
2. Entrá en: https://github.com/mauricioezequielmora-dot/mundial-2026
3. Elegí Add file > Upload files.
4. Seleccioná TODO el contenido extraído, incluidas las carpetas audio, flags e icons.
5. Verificá que index.html quede en la raíz del repositorio, no dentro de otra carpeta.
6. Mensaje sugerido: Corregir banderas y agregar radar automático
7. Elegí Commit directly to the main branch.
8. Confirmá Commit changes.
9. Esperá la tilde verde en Actions y luego abrí la página publicada.

ACTUALIZACIÓN EN EL SAMSUNG
1. Cerrá la PWA.
2. Abrí la página en Chrome y actualizá.
3. Si sigue apareciendo la versión vieja, eliminá la aplicación instalada y volvé a instalarla desde Chrome.
4. Tocá TOCÁ PARA INICIAR para habilitar pantalla completa y audio web.
5. Comprobá que cada selección muestre una bandera distinta.
6. Hacé un vivo no listado y verificá el audio desde otro dispositivo.
