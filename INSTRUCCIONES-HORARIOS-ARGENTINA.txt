CAMBIOS TÉCNICOS
- Fecha y hora guardadas directamente como hora argentina.
- Conversión manual fija a UTC-3 para el contador.
- El contador muestra horas y minutos separados con claridad.
- Se cambió la versión de caché de la PWA.
- index.html usa una dirección versionada de knockout.js para impedir que Chrome conserve el archivo anterior.

DESPUÉS DE SUBIR
1. Esperar la tilde verde en GitHub Actions.
2. Cerrar completamente la PWA en el Samsung.
3. Volver a abrirla.
4. Si persiste la versión anterior, abrir la página en Chrome y recargar una vez.
